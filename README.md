# Bella Select 🧴

Sistema de gestión para negocio de perfumes.

## Funcionalidades
- 📦 Inventario de perfumes con fotos comprimidas
- 📷 Escáner de código de barras por cámara
- 💰 Registro de ventas (contado, apartado, crédito)
- 👥 Gestión de clientes con link a WhatsApp
- 📋 Apartados con seguimiento de pagos
- 💳 Créditos con abonos
- 📊 Dashboard con estadísticas en tiempo real

## Stack
- HTML + CSS + JS vanilla (sin frameworks)
- Supabase (base de datos en la nube)

## Deploy
Subir `index.html` a Vercel como sitio estático.
